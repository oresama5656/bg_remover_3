📋 背景透過ツール開発要件定義書（イラスト特化・軽量版）
以下の要件に従って、イラスト画像の背景透過（Background Removal）ツールのPython実装コードを作成してください。過度に複雑な多段階モデルは避け、既存ライブラリを活用した軽量で保守性の高い設計を希望します。
1. プロジェクト概要
目的: イラスト画像（アニメキャラクターなど）から被写体を抽出し、主線やエッジを綺麗に残した背景透過画像（PNG）を生成するツールを開発する。
主要な課題: イラスト特有の細い線画の維持と、被写体内部の白い部分（白目や服など）が背景と誤認されて透過される「中抜け」の防止。
2. コア処理（セグメンテーション）の要件
使用ライブラリ: rembg ライブラリを使用する
。
使用モデル: rembgに内蔵されているアニメ・イラストに特化した高精度モデルである isnet-anime を指定して実装すること
。
マッティングオプション: 境界の滑らかさを調整するため、rembg.remove 実行時に alpha_matting=True を有効化できるようにし、エッジのシャープさをコントロールするパラメータ（alpha_matting_erode_structure_size など）を引数で調整可能な設計にする
。
3. ポストプロセッシング（後処理）の要件 イラスト特有のエラーを防ぐため、以下の後処理を追加してください。
中抜け（Holes）の修復: scipy.ndimage.binary_fill_holes 等のモルフォロジー演算を用いて、抽出された被写体（前景マスク）の内部に発生した意図しない透過穴を自動的に塞ぐ処理を実装する
。
4. 技術スタック
言語: Python 3.x
必須パッケージ: rembg, Pillow, numpy, scipy, opencv-python (必要に応じて)
5. 期待するアウトプット
上記を満たす単一のPythonスクリプト（指定フォルダ内の全画像を自動で一括処理するバッチ機能を含むこと）
。
実行に必要な環境構築コマンド（pip install rembg[cpu] pillow scipy など）の明記
。
出力は透明度を保持したRGBA形式の .png フォーマットで保存すること
。
